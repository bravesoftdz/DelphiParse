Uso

Acesse a pasta source, crie uma cópia do arquivo: DelphiParse.Configuration.pas.bak Renomeie a cópia para: DelphiParse.Configuration.pas Neste arquivo, informe sua APP ID e REST API Key. O projeto conta com um projeto de teste contendo um formulário que mostra algumas funções já implementadas. Além disso, conta com um projeto de teste unitário (Dunit).
